Manual Tests

Open the application below and test the following:

  1. Click the gear icon in the header - drawer should slide in from right
  2. Click outside the drawer - drawer should close
  3. Open drawer and press Escape key - drawer should close
  4. Enter an API key and click Save - should show green checkmark for 2 seconds
  5. Verify API key is masked showing only last 6 characters as "hf_...xYz123"
  6. Click Delete button - should clear the API key
  7. Verify status indicator shows red "No API Key" when no key is saved
  8. Verify status indicator shows green "API Key Saved" when key is saved
  9. Resize browser to < 480px - drawer should be full width
  10. Resize browser to > 480px - drawer should be 360px wide
